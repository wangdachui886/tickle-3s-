# Tickle 轻记 v0.1.0-alpha

这是第一个公开体验版。

Tickle 轻记是一个面向中文用户的小记账工具。它不联网、不登录、不上传，记录默认保存在本地；重点不是自动识别账单，而是让你在 App 或桌面小组件上快速记一笔。

## 这版能做什么

- App 内快速记录支出和收入
- 桌面小组件记账，支持 4x2、4x3、4x4 深色 / 浅色版本
- 流水记录按日期查看
- 日 / 月 / 年统计
- CSV 导出与最近一次备份恢复
- 自定义常用分类

## 下载说明

本次提供的是 debug / alpha 体验包，不是应用商店正式签名包。Android 安装时可能提示「未知来源」或风险提醒。

## 数据说明

记录默认保存在设备本地，不会上传到服务器。需要备份时，请在 App 内导出 CSV，并自行保存。

## 已知限制

- 不支持自动同步和多设备协作
- 不支持银行卡、账单或短信自动识别
- CSV 字段还比较少，后续会继续补充
- 不同 Android 机型的小组件显示效果可能还需要继续收集反馈

如果你想要的是自动同步、账单识别、多账户资产分析，Tickle 轻记可能不是最合适的选择。它现在只想先把一件小事做好：少打扰地记一笔。
